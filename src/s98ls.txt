***** S98 Loop Serach Support Tool "S98ls" Version 0.03 ******
						Programmed by Manbow-J

[このソフトについて]
	このソフトは、s98ファイルのループ位置の検出をサポートするプログラムです。
	設定時間から数秒の間のデータをデータ先頭から比較して同じ個所が発見されたら
	ループ位置としてファイルを出力します。

[動作環境]
	同梱の実行ファイルは、Windows2000 の コマンドプロンプトで動作を確認しています。
	また、動作させた結果に関しては、いかなる場合においても保証はいたしません
	のでご注意を。

[使い方]
	>s98ls 入力ファイル (出力ファイル) (オプション)

	とすれば、処理を開始します。
	オプションは以下のような物があります。

	-sxx ... xx秒の位置からループの検索を開始します。必ず指定してください
	-lxx ... 開始位置からxx秒の間をループ検索データとします(default:1=1秒)。
        -cxx ... 特定のチャンネルデータのみでループ位置を検索します。
                 チャンネルとパートは以下のようになっています。
                     0〜5 : FM  ch.1〜ch.6
                     6〜8 : SSG ch.1〜ch.3
        -o   ... ループ検索時のログを出力します。ログファイルは"<入力ファイル>.log"です。

	出力ファイルは省略できます。省略した場合、"<入力ファイル>.xxxx.s98"と言
	うファイル名で出力されます。
	※検索位置は必ず指定してください。

	例).
	>s98ls NP2_0000.s98 -s38 -l1
	NP2_0000.s98ファイルの38秒目から1秒間をループ検索の対象としてループ開始
	位置を探します。出力ファイル名はNP2_000.s98.xxxx.s98となります。

	ちなみに、単に
	>s98ls
	とだけ実行するとヘルプが見れるようになっています。

[ログファイル出力]
        ログファイルはs98ファイルのデータを

            時間(ms):ファイルオフセット:レジスタデータ

        という形式でテキスト出力します。どうしてもループ地点が発見できないときは
        このログからデータを追いかけると探せるかもしれません。
        ファイルオフセットの値はs98の
            0014 DWORD(LE)  FILE OFFSET TO DUMP DATA
            0018 DWORD(LE)  FILE OFFSET TO LOOP POINT DUMP DATA
        これらの値と扱いは同じです。自力でループ位置を探すときの参考にしてくだ
        さい。
